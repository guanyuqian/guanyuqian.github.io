    和为K的连续子数组

    Guanyuqian 4/22/2021 连续子数组前缀和哈希表LeetCode

    你需要找到该数组中和为 k 的连续的子数组的个数。

    # 题目描述

    给定一个整数数组和一个整数 k，你需要找到该数组中和为 k 的连续的子数组的个数。

    560. 和为K的子数组 (opens new window)

    # 示例

    输入:nums = [1,1,1], k = 2
输出: 2 , [1,1] 与 [1,1] 为两种不同的情况。

    2

    # 提示

    • 数组的长度为 [1, 20,000]。

    • 数组中元素的范围是 [-1000, 1000]，且整数 k 的范围是 [-1e7, 1e7]。

    # 解法 前缀和+哈希表

    # 求子数组和 前缀和

    通过维护数组每一项前缀和，可以在得到任意子数组的和。设前缀和为prefix[]，那么子数组nums[i:j + 1]的和为：

    # 求判断子数组和为k 哈希表

    通过遍历数组nums可以得到前缀和，和为k的子数组存在等式：

    k==prefix[j]−prefix[i−1]⇒prefix[i−1]==prefix[j]−kk == prefix[j] - prefix[i - 1] \Rightarrow prefix[i - 1] == prefix[j] - k k==prefix[j]−prefix[i−1]⇒prefix[i−1]==prefix[j]−k

    那么在遍历到任一位置j时只需要判断之前是否存在一个位置i有prefix[i]==prefix[j]−kprefix[i] == prefix[j] - kprefix[i]==prefix[j]−k 即可，判断某个值是否存在，使用哈希表可以加速操作。

    实现代码为：

    func subarraySum(nums []int, k int) (res int) {
    pre_sums, pre_sum := map[int]int{0:1}, 0
    for _, n := range nums {
        pre_sum += n
        if val, ok := pre_sums[pre_sum - k]; ok {
            res += val
        }
        pre_sums[pre_sum]++
    }
    return
}
